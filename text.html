text layout
  1. Do we have overflow? if Yes, then we have overflow rule
  2. If overflow is handled as clip, hidden
    • overflow is handled as hidden, do we need to show "ellipsis"? if yes, text-overflow:ellipsis
    • overflow is handled as clipped
  3. Do we have text wrap(white-space)
    • normal: white spaces collapse. Line break is ignored. Browser breaks line as necessary.
    • nowrap: white spaces collapse. Line break is ignored. Browser breaks line only when <br/>
    • pre: white spaces/line breaks are respected.
    • pre-line: white spaces are collapsed. Browser breaks lines on line break, <br/> and necessary
    • pre-wrap: White spaces are not collapsed. Browser breaks lines on line break, <br/> and necessary
  4. How does browser break line if needed?
    • word-wrap: normal
    • word-wrap:break-word
    • word-break: break-all|keep-all|normal, for CJK character
