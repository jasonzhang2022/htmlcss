text layout
  1. Do we have overflow? if Yes, then we have overflow rule
  2. If overflow is handled as clip, hidden, or wrap:
    • overflow is handled as hidden, do we need to show "ellipsis"? if yex, text-overflow:ellipsis
    • overflow is handled as clipped
    • overflow is handled as wrap, how do we handle word-break: overflow-wrap: break-word
  3. if no overflow, how is line break in text is handled(white-space)
  4. if line break is needed, how is word-break is handled: word-break:break-all/break-word. if not break-all, browser will adjust white space to make sure line break does not break word.
